## Installation

1. Clone the repository:
   ```
   git clone https://github.com/vkprogrammer-001/FluttyCalc.git
   ```

2. Ensure Flutter is set up:
   ```
   flutter doctor
   ```

3. Fetch dependencies:
   ```
   flutter pub get
   ```

4. Run the app:
   ```
   flutter run
   ```
